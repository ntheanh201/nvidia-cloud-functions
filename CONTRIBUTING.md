# Contributing to NVIDIA Cloud Functions

Thank you for your interest in contributing to NVCF.

## Reporting Issues

The fastest way to contribute is to [open an issue](https://github.com/NVIDIA/nvidia-cloud-functions/issues/new/choose). Please use the provided templates and be as specific as possible.

## Pull Requests

Each NVCF component lives in its own repository (see [README.md](README.md)). Pull requests should be opened against the relevant component repository, not this one.

## Code of Conduct

This project follows the [NVIDIA Open Source Code of Conduct](https://github.com/NVIDIA/.github/blob/main/CODE_OF_CONDUCT.md).
